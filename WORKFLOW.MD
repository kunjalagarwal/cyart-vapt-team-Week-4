# Full VAPT Workflow

1. Target Identification

   * Discover target IP
   * Verify host availability

2. Reconnaissance

   * Perform Nmap scanning
   * Identify exposed services

3. Vulnerability Assessment

   * Run OpenVAS scan
   * Review findings

4. Validation

   * Verify web services using Burp Suite
   * Enumerate applications

5. Analysis

   * Correlate findings
   * Assess risk levels

6. Remediation

   * Patch vulnerable services
   * Disable unnecessary services
   * Apply least privilege

7. Rescan

   * Verify remediation effectiveness

8. Reporting

   * Prepare PTES report
   * Create stakeholder summary

Conclusion:
The engagement demonstrated the complete VAPT lifecycle including reconnaissance, assessment, validation, remediation planning, and reporting within a controlled laboratory environment.
